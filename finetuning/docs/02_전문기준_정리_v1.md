# 전문 기준 정리

LLM Fine-tuning에 적용한 스포츠의학/건강과학 전문 기준을 정리합니다.

## 1. ACSM Guidelines for Exercise Testing and Prescription

### 1.1 운동 강도 분류

| 강도        | %HRmax | %HRR (여유심박수) | RPE   | 설명         |
| ----------- | ------ | ----------------- | ----- | ------------ |
| 매우 가벼움 | <57%   | <30%              | <9    | 일상 활동    |
| 가벼움      | 57-63% | 30-39%            | 9-11  | 가벼운 걷기  |
| 중강도      | 64-76% | 40-59%            | 12-13 | 빠른 걷기    |
| 고강도      | 77-95% | 60-89%            | 14-17 | 조깅, 달리기 |
| 최대        | ≥96%   | ≥90%              | ≥18   | 전력 질주    |

### 1.2 카보넨 공식 (Karvonen Formula)

목표 심박수 계산:

```
최대심박수(HRmax) = 220 - 나이
여유심박수(HRR) = HRmax - 안정시심박수
목표심박수 = (HRR × 운동강도%) + 안정시심박수
```

**예시 (30세, 안정시심박수 65bpm, 중강도 60%):**

- HRmax = 220 - 30 = 190bpm
- HRR = 190 - 65 = 125bpm
- 목표심박수 = (125 × 0.60) + 65 = 140bpm

### 1.3 운동 처방 기본 원칙 (FITT-VP)

| 요소            | 권장                   |
| --------------- | ---------------------- |
| **F**requency   | 주 3-5회               |
| **I**ntensity   | 중-고강도 (40-89% HRR) |
| **T**ime        | 회당 20-60분           |
| **T**ype        | 유산소 + 근력          |
| **V**olume      | 주 150-300분           |
| **P**rogression | 점진적 증가            |

## 3. Buchheit (2014) - HR Monitoring 연구

### 3.1 안정시 심박수 변화와 피로

| resting_heart_rate 변화 | 의미        | 권장 조치      |
| ----------------------- | ----------- | -------------- |
| ±5bpm 이내              | 정상 변동   | 계획대로 운동  |
| +5-10bpm                | 경미한 피로 | 강도 조절 고려 |
| +10-15bpm               | 급성 피로   | 저강도만 권장  |
| +15bpm 이상             | 과훈련/질병 | 휴식 필수      |

### 3.2 핵심 인사이트

> "안정시 심박수의 급격한 상승(+10bpm 이상)은 과훈련 증후군,
> 감염 초기, 또는 심각한 스트레스의 지표가 될 수 있다."

**적용:**

- 매일 아침 기상 직후 측정 권장
- 평소 대비 변화 추적이 절대값보다 중요
- +10bpm 이상 시 고강도 운동 제한

## 4. Milewski et al. (2014) - 수면과 부상 연구

### 4.1 연구 결과

| 수면 시간      | 부상 위험    |
| -------------- | ------------ |
| 8시간 이상     | 기준 (1.0배) |
| 6-8시간        | 1.2배        |
| **6시간 미만** | **1.7배**    |

### 4.2 핵심 인사이트

> "청소년 운동선수에서 8시간 미만의 수면은 부상 위험을
> 1.7배 증가시킨다."

**적용:**

- 7-9시간 수면 권장 (성인 기준)
- 5시간 미만: 고강도 운동 금지
- 6시간 미만: 중강도까지만 권장

### 4.3 수면 품질 요소

| 요소           | 권장     |
| -------------- | -------- |
| 총 수면 시간   | 7-9시간  |
| 깊은 수면 비율 | 15-20%   |
| REM 수면 비율  | 20-25%   |
| 수면 효율      | 85% 이상 |

## 5. 산소포화도 기준

### 5.1 정상 범위

| oxygen_saturation | 상태    | 조치           |
| ----------------- | ------- | -------------- |
| 95-100%           | ✅ 정상 | -              |
| 93-95%            | ⚠️ 경계 | 모니터링       |
| 90-93%            | 🚨 낮음 | 의료 상담 권장 |
| <90%              | 🚨 위험 | 즉시 의료 조치 |

### 5.2 운동 시 고려사항

- 고강도 운동 중 일시적 저하 가능
- 고지대에서 낮아질 수 있음
- 지속적 저하 시 운동 중단

## 6. BMI 기준 (아시아 기준)

### 6.1 분류

| bmi       | 분류       | 건강 위험      |
| --------- | ---------- | -------------- |
| <18.5     | 저체중     | 영양 부족 위험 |
| 18.5-22.9 | 정상       | 낮음           |
| 23-24.9   | 과체중     | 약간 증가      |
| 25-29.9   | 비만 1단계 | 증가           |
| ≥30       | 비만 2단계 | 높음           |

### 6.2 운동 처방 시 고려

| bmi 범위 | 운동 권장           |
| -------- | ------------------- |
| 저체중   | 근력 운동 중심      |
| 정상     | 균형 잡힌 운동      |
| 과체중   | 유산소 강화, HIIT   |
| 비만     | 저충격 유산소, 걷기 |
